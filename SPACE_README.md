# MindCare

An explainable AI mental health companion (DistilBERT intent classification,
LIME explanations, rule-based crisis detection, Qwen 2.5 generation).

**Not a diagnostic or clinical tool.** This is a Final Year Project
prototype, not a clinically reviewed service. If you're in immediate
danger, please contact your local emergency services or a crisis line
(the app also surfaces these directly if it detects crisis language).

See the full project README and DEPLOYMENT.md in the repository for
architecture details and setup instructions.
